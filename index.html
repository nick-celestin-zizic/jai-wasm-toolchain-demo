Simp

Simp is the simple graphics API provided with the Jai modules, but now on the web! It uses WebGL as the rendering backend.

Multiple Windows

Some applications may need to open multiple windows, so we make this work on the browser as well. Just make sure to position your windows precisely where you want them when calling create_window, because not specifying a window's position makes it occupy the full page and forwards page resizes to that window.

Input

In addition to being able to open windows, the language comes with a module for handling user input.

GetRect

GetRect is Jai's immediate mode GUI library, it includes a bunch of useful widgets and is much saner than HTML.

Sound_Player

Sound_Player is an example of a module that we have to remap completely since it is too low level and reliant on threads to be extended to work on the web like the Input module. Luckily remap_import allows us to use WebAudio as long as we write a jai wrapper that exposes the same interface as the original module.

Skeletal Animation

This is a classic example that uses OpenGL and a very particular coding style to animate a 3D model.

Invaders

A game by Jonathan Blow made to showcase various language features.
